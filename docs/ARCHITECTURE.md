## ğŸ§© Component Details

### Frontend (React)
- **State Management**: React Context + useReducer
- **Routing**: React Router v6
- **HTTP Client**: Axios with interceptors
- **UI Components**: Custom components with Tailwind CSS
- **Real-time**: Socket.io for streaming responses

### Backend (Node.js/Express)
- **API**: RESTful endpoints with OpenAPI documentation
- **File Upload**: Multer with size/type validation
- **Authentication**: JWT-based (simplified for MVP)
- **Database**: Mongoose ODM for MongoDB
- **Validation**: Joi/Zod schemas

### AI Service (Python/FastAPI)
- **Framework**: FastAPI with async support
- **PDF Processing**: PyMuPDF (fitz)
- **Embeddings**: OpenAI text-embedding-ada-002
- **Vector DB**: ChromaDB (local persistence)
- **LLM**: OpenAI GPT-4/GPT-4o via API
- **Caching**: SQLite for offline support

## ğŸ” Security Considerations

1. **API Keys**: Environment variables, never in code
2. **File Upload**: Type validation, size limits, virus scanning (future)
3. **Input Sanitization**: All user inputs validated
4. **CORS**: Configured for specific origins
5. **Rate Limiting**: Per-user request limits
